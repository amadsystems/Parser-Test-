Arithmetic Expression Parser

This project is a basic arithmetic expression parser written in Python. It takes a user input expression, tokenizes it using a lexer, and evaluates it using a recursive descent parser.

Features
Supports +, -, *, /
Handles parentheses ( )
Applies correct operator precedence
Interactive terminal input
Example
Input: 8 * 11 / 2
Result: 44.0
How to Run
python parser_test.py
Requirements
Python 3.x
